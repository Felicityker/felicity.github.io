徒步西藏攻略1（路线篇）

发表于 2019-06-09 | 更新于 2019-07-02 | 分类于 旅行游记 | 评论数： | 阅读次数：   |   字数统计: 1.6k(字)

有人说，人的一生至少要有两次冲动：
谈一场奋不顾身的爱情，去一次说走就走的旅行。
人的心总是在远方，走的越远离自己的心越近；
人的灵魂总是在天空，登的越高离自己的灵魂越近。
一个人，一条路，人在途中，心随景动，从起点，到尽头，
也许快乐，也许孤独，如果心在远方，只需勇敢前行，梦想自会引路，
有多远，走多远，把足迹连成生命线。
我说(pioneer说的)
其实梦想，并不奢侈，只要勇敢的迈出第一步。
因为有梦，所以勇敢出发，选择出发，便只顾风雨兼程。
旅行的意义，即使是同一个世界，我们发现的却是不一样的世界。
本文详细讲述徒步西藏的路线攻略，希望对你有所帮助

终点为拉萨，进藏共五条路线：
大路通途的青藏线: 109国道
最具传奇色彩的滇藏线: 214国道
最具挑战性的新藏线: 219国道
最具魅力的川藏线: 318国道 317国道
离喜马拉雅山最近的: 中尼公路

公路

连接西藏与外界的公路共有五条，分别是青藏线、川藏线、滇藏线、新藏线、中尼公路。青藏线从格尔木到拉萨是大多数人的首选，因为它是唯一一条可以直达拉萨的公路，也是五条天路中路况最好的一条。但似乎从上海到日喀则的318国道人更多诶，所以本文侧重于318国道的攻略。

航空

西藏自治区内有通航的机场五座，分别是拉萨贡嘎机场、日喀则和平机场、阿里昆莎机场、林芝米林机场和昌都邦达机场。广州、重庆、西宁、西安、上海等多个城市都有航班往返拉萨，拉萨贡嘎机场位于拉萨市区西南的甲竹林镇，距离市区大约65km。

铁路

相比于航空交通高昂的票价，铁路交通就更显得亲民。目前，北京、上海、广州、成都、兰州等地都有通往拉萨的火车，从成都途经广元、宝鸡、兰州、西宁、格尔木、那曲等地到达拉萨。

骑行

拉萨、日喀则等大城市是提供维修和补给的中转站，除此之外的县、镇几乎无法保养和维修自行车，最好在出发地就准备好自行车的换补零件。在拉萨的各个青年旅社也可以租到自行车，方便在市区自由行走。

徒步

青藏线（109国道）

青藏线起于青海省的首府西宁，止于拉萨，是世界上最长的柏油公路，五条线路中海拔最高，进藏里程最短，路况最好，四季皆可通行。
里程：
$格尔木\xrightarrow{\ \ \ \ 269km\quad} $五道梁$\xrightarrow{\ \ \ \ 150km\quad} $沱沱河$\xrightarrow{\ \ \ \ 110km\quad} $雁石坪$\xrightarrow{\ \ \ \ 100km\quad} $唐古拉山$\xrightarrow{\ \ \ \ 89km\quad} $安多$\xrightarrow{\ \ \ \ 138km\quad} $那曲$\xrightarrow{\ \ \ \ 164km\quad} $当雄$\xrightarrow{\ \ \ \ 75km\quad} $羊八井$\xrightarrow{\ \ \ \ 87km\quad} 拉萨$

滇藏线（214国道）

滇藏公路全长1930千米，始于云南南部的西双版纳景洪，止于青藏交界的多普玛，后经西藏的芒康与川藏南线（318国道）交汇，直达拉萨这是一条因茶叶而闻名的古道，一般游客会从昆明出发开启滇藏之旅。
里程：
$昆明\xrightarrow{\ \ \ \ 397km\quad} $下关$\xrightarrow{\ \ \ \ 14km\quad} $大理$\xrightarrow{\ \ \ \ 166km\quad} $丽江$\xrightarrow{\ \ \ \ 145km\quad} $中甸$\xrightarrow{\ \ \ \ 152km\quad} $德钦$\xrightarrow{\ \ \ \ 103km\quad} $盐井$\xrightarrow{\ \ \ \ 111km\quad} $芒康(后接318国道)

新藏线（219国道）

浩瀚无垠，一望无边，是这条公路的真实写照。相比于其他四条公路的景象，新藏线如同进入了无人区，湛蓝的天空下，圣山冈仁波齐和喀喇昆仑可能是你行进路上唯一的伙伴了，前不久就有人在这条路上失联，所以最好别走这条线。
里程：
$叶城\xrightarrow{\ \ \ \ 69km\quad} $菩萨$\xrightarrow{\ \ \ \ 88km\quad} $库地达坂$\xrightarrow{\ \ \ \ \ ······\quad} $拉萨（不希望读者走这条线路，故省略里程······）

川藏南线（318国道）

这是一条每个旅游者一生至少要全程穿越过一次的公路，对于那些向往自由超脱的旅游者，这里有你理想的“天国”。小心你的灵魂在这条无尽蜿蜒的天路上会不知不觉地跃出身体，让你的思绪永远沉醉其间。
里程：
$成都\xrightarrow{\ \ \ \ 139km\quad} $雅安$\xrightarrow{\ \ \ \ 168km\quad} $泸定$\xrightarrow{\ \ \ \ 49km\quad} $康定$\xrightarrow{\ \ \ \ 75km\quad} $新都桥$\xrightarrow{\ \ \ \ 74km\quad} $雅江$\xrightarrow{\ \ \ \ 143km\quad} $理塘$\xrightarrow{\ \ \ \ 165km\quad} $巴塘$\xrightarrow{\ \ \ \ 36km\quad} $竹芭笼$\xrightarrow{\ \ \ \ 71km\quad} $芒康$\xrightarrow{\ \ \ \ 158km\quad} $左贡$\xrightarrow{\ \ \ \ 107km\quad} $邦达$\xrightarrow{\ \ \ \ 94km\quad} $八宿$\xrightarrow{\ \ \ \ 90km\quad} $林芝$\xrightarrow{\ \ \ \ 19km\quad} $八一$\xrightarrow{\ \ \ \ 127km\quad} $工布江达$\xrightarrow{\ \ \ \ 206km\quad} $墨竹工卡$\xrightarrow{\ \ \ \ 68km\quad} $拉萨

川藏北线（317国道）

里程：
$成都\xrightarrow{\ \ \ \ 54km\quad} $都江堰$\xrightarrow{\ \ \ \ 83km\quad} $卧龙$\xrightarrow{\ \ \ \ 148km\quad} $日隆$\xrightarrow{\ \ \ \ 43km\quad} $小金$\xrightarrow{\ \ \ \ 55km\quad} $丹巴$\xrightarrow{\ \ \ \ 77km\quad} $八美$\xrightarrow{\ \ \ \ 68km\quad} $道孚$\xrightarrow{\ \ \ \ 72km\quad} $炉霍$\xrightarrow{\ \ \ \ 97km\quad} $甘孜$\xrightarrow{\ \ \ \ 95km\quad} $玛尼干戈$\xrightarrow{\ \ \ \ 112km\quad} $德格$\xrightarrow{\ \ \ \ 24km\quad} $金沙江大桥$\xrightarrow{\ \ \ \ 85km\quad} $江达$\xrightarrow{\ \ \ \ 228km\quad} $昌都$\xrightarrow{\ \ \ \ 248km\quad} $丁青$\xrightarrow{\ \ \ \ 225km\quad} $巴青$\xrightarrow{\ \ \ \ 136km\quad} $那曲$\xrightarrow{\ \ \ \ 164km\quad} $当雄$\xrightarrow{\ \ \ \ 75km\quad} $羊八井$\xrightarrow{\ \ \ \ 87km\quad} $拉萨
